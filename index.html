Hill Nodebox
for Minetest

Craft and place hill-like nodes.

Download GitHub Minetest CoRNeRNoTe

Features

  • Hill-like nodes using nodebox.
  • Comprised of 16 levels of square nodeboxes.

Docs

Requirements

  • Minetest >=0.4.3

Installation

Download and uncompress the latest version, and move the the minetest-hill_nodebox-master folder into your minetest/mods folder.
